# Xiaomi 17 Ultra Color Match Studio

샤오미 17 울트라에서 원하는 색감을 빠르게 찾고, 비교하고, 저장할 수 있는 웹 기반 컬러 매칭 & 룩 튜닝 스튜디오입니다.

**Xiaomi 17 Ultra Color Match Studio**는  
사용자가 원하는 색감의 이미지를 업로드하면, 그 톤을 분석하여 **샤오미 17 울트라 카메라에서 유사한 결과를 얻기 위한 스타일, 필터, 수동 튜닝 값**을 추천해주는 웹 기반 색감 매칭 툴입니다.

이 툴은 단순히 이미지를 분석하는 데서 끝나지 않고,  
**AI 기반 추천 → 프리뷰 비교 → 수동 튜닝 → 프리셋 저장**까지 한 번에 가능한 구조로 설계되었습니다.

특히 다음과 같은 목적에 적합합니다.

- 원하는 참고 사진의 색감을 샤오미 17 울트라에서 최대한 비슷하게 맞추고 싶은 경우
- Leica 스타일, 샤오미 기본 필터, 커스텀 수치를 함께 조합해 보고 싶은 경우
- vivo, Fujifilm, Ricoh GR, Hasselblad 등 대표 색감 방향을 샤오미 기반으로 근사 재현하고 싶은 경우
- 사진 결과를 빠르게 비교하고 자신만의 프리셋으로 저장하고 싶은 경우

## 주요 기능

- 타겟 이미지 업로드 후 색감 분석
- Leica 스타일 추천
  - Leica Authentic
  - Leica Vibrant
- 샤오미 필터 추천 및 우선순위 제안
- 수동 튜닝 지원
  - 차가움 / 따뜻함
  - 청록 / 자홍
  - 채도
  - 대비
  - 선명도
  - EV
- 실시간 적용 프리뷰
- 추천 썸네일 비교
- 내장 룩 프리셋 제공
  - Fujifilm Film Simulation 계열
  - Ricoh GR Positive
  - Hasselblad 계열
  - vivo ZEISS 계열 등
- 프리셋 저장 / 불러오기 / 즐겨찾기
- 간편 모드 / 전문가 모드 지원

## 특징

이 프로젝트는 샤오미 17 울트라의 실제 카메라 UX를 고려해 설계되었으며,  
**“AI가 정답을 강요하는 툴”이 아니라, AI 추천을 시작점으로 사용자가 직접 미세 조정할 수 있는 색감 에디터**를 목표로 합니다.

즉,  
**분석기 + 프리뷰 툴 + 실시간 튜닝 에디터 + 프리셋 매니저**의 성격을 함께 갖는 프로젝트입니다.

## 주의사항

- 본 프로젝트는 샤오미 카메라 앱의 내부 ISP/이미지 파이프라인을 그대로 복제한 툴이 아닙니다.
- 실제 촬영 결과는 광원, 노출, 피사체, 렌즈 화각, HDR 동작 등에 따라 달라질 수 있습니다.
- 따라서 이 툴은 **실전 촬영을 위한 고품질 근사값 추천 도구**로 사용하는 것이 가장 적절합니다.

# Xiaomi 17 Ultra Color Match Studio

A web-based color matching and look tuning studio for Xiaomi 17 Ultra, designed to analyze reference images, preview results, and build reusable presets.

**Xiaomi 17 Ultra Color Match Studio** is a web-based color matching tool that analyzes a reference image and suggests the closest possible **style, filter, and manual tuning values** for recreating a similar look on the **Xiaomi 17 Ultra** camera.

This project goes beyond simple image analysis.  
It is designed as a complete workflow:

**AI-assisted analysis → preview comparison → live manual tuning → preset saving**

It is especially useful for users who want to:

- recreate the mood and tone of a target image on Xiaomi 17 Ultra
- experiment with Leica styles, Xiaomi filters, and custom tuning values
- approximate signature looks inspired by vivo, Fujifilm, Ricoh GR, Hasselblad, and similar systems
- compare results quickly and save their own reusable presets

## Key Features

- Target image upload and color analysis
- Leica style recommendation
  - Leica Authentic
  - Leica Vibrant
- Xiaomi filter recommendation with ranked priority
- Live manual tuning
  - Cool / Warm
  - Teal / Magenta
  - Saturation
  - Contrast
  - Sharpness
  - EV
- Real-time preview comparison
- Ranked thumbnail suggestions
- Built-in look presets
  - Fujifilm film simulation inspired presets
  - Ricoh GR Positive inspired preset
  - Hasselblad-inspired look
  - vivo ZEISS-inspired looks
- Preset saving / loading / favorites
- Simple mode / Pro mode

## Philosophy

This tool is designed around the real-world camera workflow of Xiaomi 17 Ultra users.

Rather than acting like an “AI that gives one final answer,” it is built as a **look exploration and editing studio**, where AI provides a strong starting point and the user refines the result interactively.

In that sense, the project is a combination of:

**an analyzer + preview tool + live tuning editor + preset manager**

## Disclaimer

- This project does **not** replicate Xiaomi’s internal ISP or camera pipeline exactly.
- Final real-world results may vary depending on lighting, exposure, subject, focal length, and HDR behavior.
- It should be used as a **high-quality practical approximation tool** for look matching and tuning.
